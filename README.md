# Inflation Roulette: Reloaded
_Inflation Roulette: Reloaded_ is a luck-based strategy game developed by NicklySuffer.

Aiming to expand upon [a game of the same title](https://snowyboiwhomakesstuf.itch.io/inflation-roulette), it features more characters, more cosmetic elements, skill-based gameplay, and modding support.

This game is developed using HaxeFlixel, officially supporting Windows, Android, and Linux targets.

## FAQ
### Troubleshooting
**Issue: The game keeps crashing without a crash message.**

**Resolve:** Please ensure that your system meets the minimum specifications.

**Minimum specs:**
- Windows x64 / Linux x64 / Android ARM64
- 600 MHz CPU
- 200 MHz GPU
- 4 GiB RAM
### FAQ
**Q: How do I open this game?**<br>
**A:**
1. Download the package for the respective platform you're on.
2. Extract the file's contents.
3. Run ``Inflation-Roulette.exe``. *Do **NOT** isolate other folders in the extracted folder from the executable.*

**Q: How do I install addons?**<br>
**A:**
1. Place the addon package into the `addons/` folder in the game's root directory.
2. Extract the package. Make sure that the folder's first subdirectory contains `metadata.json`.
3. Restart Inflation Roulette: Reloaded.

### Others
**Q: How can I contribute to translations?**<br>
**A:** Open a pull request on this GitHub repository.
1. Create a fork of the repository.
   1. Click on **Fork**.
   2. Click on **Create fork**.
   3. From `assets/lang/`, copy `en-US.json` and the `en-US/` folder.
   4. Rename the copied `en-US` file and folder to your language's locale code. (formatted as `language-REGION`)
   5. Edit the `.json` file's strings. Do *NOT* edit the keys.
   6. In `<language-REGION>/metadata/`, edit `metadata.json`.
2. Click the **Pull Requests** tab in the repository's menu navigation bar.
3. Click **New Pull Request**.
4. Select the **main** branch.
5. Click **Create pull request**.

**Q: May I suggest some features to the game?**<br>
